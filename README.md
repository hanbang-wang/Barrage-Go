# Barrage

一个开源的b站直播间弹幕姬，主播可以获取直播间内弹幕信息且可以进行发送弹幕功能。

从 [OctavianLee/Barrage](https://github.com/OctavianLee/Barrage) 使用 Go 重构。

## 功能介绍

+ 在直播间接收弹幕
+ 在直播间发送弹幕
+ ...

## 使用说明

+ 安装：

```bash
go get -u github.com/hanbang-wang/Barrage-Go/cmd/barrage
```

+ 代码使用：

```bash
barrage
```
